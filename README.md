#Calculadora em Python + HTML
Este é um projeto simples de calculadora feita com Python (Flask) no back-end e HTML no front-end.
Ele permite realizar operações matemáticas básicas: soma, subtração, multiplicação e divisão.

#Tecnologias usadas
- Python3
- Flask
- HTML

#Funcionalidades
- Inserir dois números
- Escolher a operação desejada (+, -, x, ÷)
- Exibir o resultado do cálculo

  #Projeto desenvolvido como prática de Python (Flask) + HTML
